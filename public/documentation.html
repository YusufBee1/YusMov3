Yusuf's Movie API (YusMov)

About: This REST API provides movie information, user management, and the ability for users to save favorite movies.

API Endpoints

Request URL Description Query Params Request Body Response
GET /movies Returns a list of all movies. None None JSON array of movie objects, e.g.:
[
  {
    "title": "Inception",
    "director": "Christopher Nolan",
    "year": 2010
  },
  ...
]
GET /movies/:title Returns data about a single movie by title. URL param: title (string) None JSON object of movie details, e.g.:
{
  "title": "Inception",
  "description": "...",
  "genre": "Sci-Fi",
  "director": "Christopher Nolan",
  "imageURL": "http://...",
  "featured": true
}
GET /genres/:name Returns data (description) about a genre by name. URL param: name (string) None JSON object with genre info, e.g.:
{
  "name": "Thriller",
  "description": "Thrillers are suspenseful and exciting..."
}
GET /directors/:name Returns data about a director by name (bio, birth year, death year). URL param: name (string) None JSON object with director info, e.g.:
{
  "name": "Christopher Nolan",
  "bio": "British-American filmmaker...",
  "birthYear": 1970,
  "deathYear": null
}
POST /users Registers a new user. None JSON object with user info:
{
  "username": "johndoe",
  "email": "john@example.com",
  "password": "password123"
}
 Confirmation message or created user object.
Example: { "message": "User johndoe registered successfully." }
PUT /users/:username Updates an existing user’s username. URL param: username (string) JSON object:
{
  "username": "newUsername"
}
 Confirmation message.
Example: { "message": "Username updated to newUsername." }
POST /users/:username/movies/:movieId Adds a movie to a user’s list of favorites. URL params: username, movieId None Confirmation message.
Example: { "message": "Movie 123 added to johndoe's favorites." }
DELETE /users/:username/movies/:movieId Removes a movie from a user’s list of favorites. URL params: username, movieId None Confirmation message.
Example: { "message": "Movie 123 removed from johndoe's favorites." }
DELETE /users/:username Deregisters an existing user. URL param: username (string) None Confirmation message.
Example: { "message": "User johndoe deregistered successfully." }

How to Use

Start the project with node index.js and visit http://localhost:8080.
